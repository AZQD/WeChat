### 1.微信小程序图片设置圆角会先闪一下才变成圆角:
```
尝试加一下这个试试，transform变形
当我们通过某些行为触发页面进行大面积绘制的时候，浏览器由于没有事先准备，应付渲染够呛，于是掉帧，于是卡顿。而will-change则是真正的行为触发之前告诉浏览器：“我待会儿就要变形了”。于是乎，浏览器把GPU给拉上了，从容应对即将到来的变形。
image{
will-change: transform
}
```
